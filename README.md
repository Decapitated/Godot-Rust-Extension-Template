# Rust GDExtension Template

<img src="https://img.shields.io/badge/Godot-v4.6-blue?labelColor=blue&logo=godotengine&logoColor=white">

A cargo-generate template for a GDExtension written in Rust using [gdext](https://github.com/godot-rust/gdext).

## Usage

Install [cargo-generate](https://github.com/cargo-generate/cargo-generate).
```
cargo install cargo-generate
```

Run cargo-generate.
```
cargo generate --git https://github.com/Decapitated/Godot-Rust-Extension-Template.git
```
